# Test Directory for AI Content Generation Factory

This directory contains test outputs and examples from the modular AI content generation system.

## Generated Content Structure

When you generate a character using the new modular system, it creates a directory with the following structure:

```
test/
└── CharacterName_YYYYMMDD_HHMMSS/
    ├── character.json          # Character data in JSON format
    ├── lore.md                # Character lore/story
    ├── image_prompt.txt       # Prompt used for image generation
    └── profile_image.webp     # Generated profile image
```

## Running Tests

### Test the Character Generator

```bash
# From the gen/ directory
python test_character_generator.py
```

### Test Individual Components

```python
from character_generator_v2 import CharacterGeneratorV2

# Create generator instance
generator = CharacterGeneratorV2("test")

# Test character JSON generation
character_json = await generator.generate_character_json(
    "Aragorn", 
    "A ranger from the north"
)

# Test character lore generation
lore = await generator.generate_character_lore(character_json)

# Test profile image generation
image_url = await generator.generate_profile_image(character_json)
```

### Generate a Complete Character

```python
from character_generator_v2 import generate_character_async

# Generate a complete character
character_dir = await generate_character_async(
    "Aragorn",
    "A ranger from the north, heir to the throne of Gondor",
    output_dir="test"
)
```

## Key Features of the Modular System

1. **Generic Building Blocks**: Uses `text_generator.py` and `image_generator.py` for AI operations
2. **Template System**: Uses `prompt_loader.py` for prompt management
3. **Schema Validation**: Uses `schema_loader.py` for data validation
4. **Modular Design**: Easy to extend and modify individual components
5. **Error Handling**: Comprehensive error handling and logging
6. **Progress Callbacks**: Optional progress tracking for long operations

## File Descriptions

- **character.json**: Complete character data following the character schema
- **lore.md**: Rich character backstory and lore generated by AI
- **image_prompt.txt**: The exact prompt used to generate the profile image
- **profile_image.webp**: Generated character profile image

## Next Steps
