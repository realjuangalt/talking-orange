# Talking Orange AR Project

An augmented reality evangelism tool for Bitcoin that brings a 3D talking orange character to life through QR code scanning and voice interaction.

## ğŸ¯ Project Overview

This project creates an AR experience where users:
1. Scan a QR code on a Bitcoin-themed image
2. Get redirected to a mobile-optimized website
3. Grant camera and microphone permissions
4. Point their camera at the image to see a 3D talking orange
5. Interact with the character through voice commands about Bitcoin
